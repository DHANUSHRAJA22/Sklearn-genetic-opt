Contributing
============

We welcome contributions to expand this collection of external references! Here's how you can contribute:

**Adding External References:**

1. **Fork the repository** and create a new branch for your contribution
2. **Add your reference** to the appropriate section above:

   - **Articles**: Blog posts, Medium articles, or published papers using sklearn-genetic-opt
   - **Tutorials and Examples**: Step-by-step guides or code examples
   - **Use Cases**: Real-world applications or case studies

3. **Format requirements**:

   - Use reStructuredText (RST) format: ``* `Title <URL>`_``
   - Ensure the title is descriptive and accurate
   - Verify all links are working and publicly accessible
   - Include the author's name in the title if it's not a well-known publication

4. **Quality guidelines**:

   - Content should demonstrate practical usage of sklearn-genetic-opt
   - Tutorials should include code examples or clear explanations
   - Articles should be well-written and technically accurate
   - Avoid promotional or purely commercial content
